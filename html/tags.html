1. Simple/universal (no specific meaning)

Used as “boxes” or for minor formatting:

  • <div> — a block-based universal container, simply a box for grouping elements.
  • <span> — a line container that highlights a meaningless piece of text.
  • <br> — line break.
  • <hr> — horizontal line, separator between blocks.

2. Semantic structural (page framework)

They make it clear which part of the document it is:

  • <header> — the top part of a page or section (header).
  • <footer> — the bottom part of a page or section (footer).
  • <main> — main content of the page.
  • <nav> — navigation block (menu).
  • <section> — окремий смисловий розділ сторінки.
  • <article> — independent material (article, news item, post).
  • <aside> — додатковий вміст, зазвичай збоку.

3. Semantic content (within the content)

Indicate the meaning of text or data:

  • <h1>-<h6> — headings from main to subheadings.
  • <p> — paragraph of text.
  • <strong> — important text (bold).
  • <em> — emphasized text (italics).
  • <blockquote> — long quotation.
  • <q> — short quotation in the text.
  • <address> — author's contact details.
  • <time> — date or time.
  • <data> — machine-readable data.
  • <code> — code fragment.
  • <pre> — preformatted text (with spaces and indents).
  • <samp> — example of program output.
  • <kbd> — text that the user must enter from the keyboard.
  • <ul> — unordered list.
  • <ol> — ordered list.
  • <li> — list item.
  • <dl> — definition list.
  • <dt> — term in the list.
  • <dd> — explanation of the term.

4. Media and interactive

These tags are designed for embedding images, audio, video, and creating interactive elements.

  • <img> — image.
  • <a> — hyperlink.
  • <video> — video file.
  • <audio> — audio file.
  • <source> — source for media.
  • <figure> — media with caption.
  • <figcaption> — caption under an image or video.
  • <details> — hidden content that can be revealed.
  • <summary> — title for the <details>block.
  • <iframe> — embedded third-party content (e.g., YouTube video).

5. Forms

Form tags are used to create data entry fields that allow users to interact with a web page.

  • <form> — form for entering data.
  • <input> — input field (text, number, password, etc.).
  • <textarea> — multi-line text field.
  • <select> — drop-down list.
  • <option> — element in a drop-down list.
  • <label> — label for an input field.
  • <fieldset> — group of fields in a form.
  • <legend> — name of a group of fields.
  • <button> — button.
